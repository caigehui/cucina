# daily-report 每日美股研报模块

一次性把四个模块的产出汇总成**一份可读、分层、客观**的《每日美股研报》,**保存为文件**到项目文件夹(供用户随时打开)。始终简体中文。

## 运行时机与定位
- 默认按 `America/New_York` 时区在美股常规交易日运行两次:
  - **美东 07:30**:常规开盘 09:30 前 2 小时,输出**「隔夜复盘 + 当日盘前展望」**研报。
  - **美东 18:00**:常规收盘 16:00 后 2 小时,输出**「当日收盘复盘 + 盘后跟踪」**研报。
- 盘前版回顾上一交易日美股收盘、扫描隔夜要闻,预览当日交易和未来事件日历;盘后版复盘当日收盘、盘后新闻/财报/指引、盘后异动和次日待验证事项。
- 周末/美股休市或无新交易日时,自动转为**「休市复盘 + 下个交易日展望」**轻量版。

## 生成流程(顺序执行)

1. **定时间**:bash 取 `TZ='America/New_York' date` 与 `TZ='Asia/Shanghai' date`,确定上一交易日与当日。
2. **宏观与事件**:按 `macro-events` 流程采集货币政策/通胀就业/财政债务/贸易政策/地缘新闻 + 未来 1-2 周事件日历(FOMC、CPI/PCE、NFP、四巫日、国债拍卖、重点财报)。给已定价 vs 未定价判断。
3. **社媒情绪**:按 `wsb-sentiment` 流程取 WSB + X 当日最热话题、散户情绪方向与 Hype 阶段、情绪温度与反向信号。
4. **科技股/赛道**:按 `tech-research` 方法论,挑当日**最值得讲的 1-3 个科技/AI 供应链看点**(财报、卡点验证信号、出口管制传导等),给结论先行的简析;不堆砌,只讲有信息增量的。
5. **持仓提示**(若用户持仓可得):按 `portfolio-advisor`,只读取持仓,给当日需关注的持仓事件与情景预案要点;**绝不下单**。无持仓数据则跳过或给"如需持仓分析请安装官方长桥技能(见 https://open.longbridge.com/skill/install.md)并完成 OAuth 授权,或粘贴持仓"。
6. **独立复核**:落盘前快速自检——数字有无来源/as-of?有无把推测当事实?有无变成买卖指令(应改为客观分析)?
7. **写文件**:见下「输出」。

## 输出(必做)

写一份 Markdown 文件到项目文件夹根目录:
`C:\Users\Administrator\Claude\Projects\美股投研\美股研报_YYYY-MM-DD.md`
(bash 路径 `/sessions/zen-wizardly-bardeen/mnt/美股投研/美股研报_YYYY-MM-DD.md`)

文件结构:

```markdown
# 美股每日研报｜YYYY-MM-DD（北京/美东）

## 0. 一句话定调
<risk-on / risk-off / 震荡分歧> + 当日最大矛盾 + 盘前关注点

## 1. 隔夜复盘
- 三大指数/Mag7/关键板块收盘、涨跌主因
- 美债/美元/油金铜/BTC/VIX 读数

## 2. 隔夜与盘前要闻（分层：货币/通胀就业/财政/贸易政策/地缘）
（每条：事实 → 已定价程度 → 边际含义；标 已证实/传闻/推测）

## 3. 今日事件日历（美东+北京）
- 当日：数据/财报/讲话 + 共识与阈值 + 方向情景
- 未来 1-2 周高亮：FOMC / CPI / NFP / 四巫日 / 国债拍卖等

## 4. 社媒情绪（WSB + X）
- 最热话题、散户押注方向、Hype 阶段、情绪温度(0-100)、反向信号

## 5. 科技股/AI 供应链看点（1-3 条，结论先行）
- 卡点 thesis / 催化 / 证伪点；标信念档与证据成色

## 6. 持仓提示（如适用，只读不下单）
- 今日需盯的持仓事件 + 情景预案要点

## 7. 风险与证伪
- 什么会推翻今天的定调

---
⚠️ 本研报为客观分析与信息整理，非投资建议。数据可能有误，决策请自行核实（DYOR）。
```

落盘后,用 `mcp__cowork__present_files` 把研报文件呈现给用户,正文只给一两句当日要点摘要。

## 约束
- ✅ 简体中文;结论先行;数字标来源 + as-of;区分 事实/传闻/推测。
- ❌ 不给买卖指令、不预测精确点位、不下单。
- 控制长度:重信息增量,不为凑字数堆砌。
