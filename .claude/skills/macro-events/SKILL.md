# macro-events 宏观事件与市场情绪模块

职责:**采集宏观与地缘新闻 → 对齐未来事件日历 → 对市场情绪与未来走向做客观分层分析**。始终用简体中文输出。

⚠️ 这是**客观分析**,不是买卖建议;不预测点位,只拆"市场在定价哪一层预期、哪些事件可能改变它"。

## 一、数据采集

### 1. 宏观/地缘新闻源(用 WebSearch + web_fetch;客户端渲染页改用 Claude in Chrome)
- **货币政策**:美联储声明/点阵图/官员讲话、FOMC 纪要、CME FedWatch 隐含降息概率。
- **通胀与就业**:CPI、PCE、PPI、非农(NFP)、失业率、JOLTS、初请失业金。
- **财政/债务**:美国预算与赤字、债务上限、国债拍卖与发行节奏、TGA/逆回购、政府关门风险(可读 Polymarket 概率)。
- **贸易/产业政策**:关税、出口管制(尤其半导体/关键材料,与 tech-research 卡点链联动)、CHIPS Act。
- **地缘政治**:冲突/制裁/能源/航运/选举,按 §三 的「多资产分层」拆解。
- **市场内部结构**:VIX、美债收益率曲线(2s10s)、美元指数 DXY、信用利差、原油/黄金/铜、比特币。

### 2. 社媒情绪(可选,与 wsb-sentiment 模块互补)
X/Twitter 财经 KOL 与 r/wallstreetbets 的宏观叙事。若已连接 Reddit/X 或 LunarCrush MCP 优先用;否则 web 抓取。把社媒当**情绪温度计与反向信号**,不当事实源。

## 二、未来事件日历(必出)

运行时先确定当前美东时间(bash `TZ='America/New_York' date`),向前看 1-2 周,列出:

| 类型 | 事件 | 为什么重要 |
|---|---|---|
| 货币政策 | FOMC 决议/纪要/官员讲话、鲍威尔证词 | 利率路径重定价,影响估值分母 |
| 数据 | CPI、PCE、NFP、零售、ISM、密歇根信心 | 通胀/增长预期,左右降息押注 |
| 衍生品 | **四巫日(quad witching,3/6/9/12 月第三个周五)**、月度 OPEX、0DTE 集中到期 | gamma/对冲流引发波动放大,方向常失真 |
| 供给 | 国债拍卖(尤其长债)、再融资公告 | 利率与流动性冲击 |
| 财报 | Mag7 / 重点持仓 / 板块龙头财报日 | 个股与板块情绪引爆点 |
| 事件 | 政府关门截止、债务上限、选举/峰会、出口管制听证 | 尾部风险与主题催化 |

每个事件标注:**日期(美东+北京双时区)、市场共识预期、关键阈值、若超/不及预期的方向含义**。

## 三、分析方法(客观、分层)

1. **区分已定价 vs 未定价**:市场定价的是**预期/感知**,不是现实。先问"这件事市场已经 price in 多少"(看 FedWatch/Polymarket/远期/IV),再判断边际惊喜在哪。
2. **指标异常先找机制根因**:盘面与新闻表面相反时,不信表面叙事,溯源真正的机制(流动性/保证金/对冲/仓位)。
3. **地缘/政策事件 → 多资产分层拆解**:不要把事件压成"战争=油"一句话。拆成多条互不相关的资产线(能源/化肥/防务/避险/利率/汇率/加密),每条问"大众怎么做 vs 客观该怎么看"。
4. **传导到持仓/赛道**:把宏观结论显式接到用户关注的板块(尤其 AI 供应链卡点链——出口管制、关税、利率对小盘成长股的久期效应)。
5. **情绪温度**:综合 VIX、put/call、AAII/WSB 情绪、资金流,给一个 risk-on / risk-off / 分歧 的判定,并标"是否处于情绪极端(反向信号)"。

## 四、输出格式(强制)

```text
🌐 宏观情绪与事件展望｜<日期，美东/北京>

① 一句话定调:<risk-on / risk-off / 震荡分歧> + 主导矛盾

② 隔夜/近端宏观要闻(分层)
- 货币政策:
- 通胀/就业:
- 财政/债务:
- 贸易/产业政策:
- 地缘:
（每条:事实 → 已定价程度 → 边际含义；标注 已证实/传闻/推测）

③ 未来 1-2 周事件日历(表，见 §二)
- 重点高亮:本期最可能引发波动的 1-3 个事件 + 阈值 + 方向情景

④ 市场内部结构读数
- VIX / 2s10s / DXY / 油金铜 / BTC：当前值 + 一句解读

⑤ 情绪温度 & 反向信号
- 当前位置(0=极度恐慌 ↔ 100=极度贪婪)、是否极端

⑥ 对赛道/持仓的传导(接 tech-research / portfolio-advisor)
- 宏观结论如何影响 AI 供应链/利率敏感成长股/用户关注板块

⑦ 证伪与盲点
- 什么数据/事件会推翻上面的定调
```

## 五、约束
- ✅ 仅简体中文;明确区分 事实/传闻/推测;数字给来源 + as-of。
- ❌ 不给买卖建议、不预测精确点位、不渲染恐慌。
- 受众:懂金融、看美股的人。
